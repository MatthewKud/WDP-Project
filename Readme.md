# Readme for JOURNALING 3005

## Description
Journaling 3005 is a web application that allows users to create 
and store personal journal entries organized by category.

## Business Rules
`<USER> <may> <write> <any number> <JOURNAL_ENTRY>` 

`<JOURNAL_ENTRY> <must> <be written by> <exactly one> <USER>`


`<JOURNAL_ENTRY> <must> <belong to> <exactly one> <CATEGORY>`

`<CATEGORY> <may> <contain> <any number> <JOURNAL_ENTRY>`
